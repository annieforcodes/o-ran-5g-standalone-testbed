This is a software simulation of a 5G network using Open RAN.
The setup was done using OpenAirInterface.
No real hardware was used, the channel was simulated via rfsim.
UDP performance analysis was done using iperf3 by varying the bitrates and measuring throughtput for both uplink and downlink.
TCP analysis was also done using iperf3  for both uplink and downlink.
ICMP ping testing was done to measure latency.
TCP throughput test using four parallel streams was performed.
Extended duration UDP testing was conducted to evaluate long-term session stability.
